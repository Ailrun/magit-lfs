## Magit Binding

-   `&`
    Initial commands for magit-lfs.
    -   `f`
        git lfs fetch
    -   `F`
        git lfs Pull
    -   `i`
        git lfs install
    -   `l`
        git lfs logs
    -   `P`
        git lfs push
    -   `U`
        git lfs update
    -   `!`
        git lfs fsck
